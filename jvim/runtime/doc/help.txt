jvim: The Jenova Cognitive Editor

jvim is the professional frontend for the Jenova Cognitive Architecture. 
It integrates traditional modal editing with a modern, AI-augmented project 
flow ("pflow"). 

If you are new to the bridge, we recommend starting with the Flight School:
>vim
    :Tutor
<
For a high-level overview of the system, see |jvim-start|.

(To follow a link, move the cursor to it and hit CTRL-]. Return with CTRL-O.)
